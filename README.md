# Deadwood
Deadwood design on console implementation

Name: Jasmine Lofton & Austin Onnsy & 

## Purpose 
 Deadwood is an interactive board game now implemented as a virtual console played adventure. 

## Compile
Compile the program by navigating to the project directory in a terminal and running javac *.java, then start the game by running java Main (replace Main with the name of the class that contains the main method).
